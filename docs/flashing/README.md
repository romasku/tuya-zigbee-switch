## Quick Reference

### Silicon Labs (ZS3L / EFR32MG21)

Most Tuya Zigbee switches use this chip. Two flashing methods available:

- **[J-Link v9](silabs_jlink_v9.md)** - Recommended, easier setup
- **[Raspberry Pi GPIO](silabs_raspberry_pi.md)** - No programmer needed

### Telink (TLSR8258)

Used in some older devices. Requires Telink-specific tools.

### ZBMINIL2

Sonoff mini relay module with custom flashing procedure.
