# Silicon Labs (EFR32MG21) Flashing

Guides for flashing custom firmware onto Silicon Labs EFR32MG21 chips (Tuya ZS3L module).

## Flashing Methods

| Method | Hardware Required |
|--------|-------------------|
| [J-Link v9](silabs_jlink_v9.md) | J-Link programmer |
| [Raspberry Pi GPIO](silabs_raspberry_pi.md) | Raspberry Pi 3B+ |

## ZS3L Module

- Chip: EFR32MG21A020
- Flash: 768 KiB
- Pinout: [Tuya ZS3L Documentation](https://developer.tuya.com/en/docs/iot/zs3l?id=K97r37j19f496)
